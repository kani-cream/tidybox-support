## 日本語

**TidyBox** は macOS 向けのファイル自動整理アプリです。メニューバーに常駐し、指定フォルダを監視してルールに基づいてファイルを自動的に整理します。

## English

**TidyBox** is a macOS file auto-organization app. It resides in the menu bar, monitors designated folders, and automatically organizes files based on rules.
