# Privacy Policy / プライバシーポリシー

Last updated: 2026-03-22

[日本語](#日本語) | [English](#english)

---

## English

### Overview

TidyBox is a macOS file auto-organization application that runs entirely on your device. We are committed to protecting your privacy.

### Data Collection

**TidyBox does not collect, transmit, or store any personal data.** Specifically:

- **No analytics**: We do not use any analytics or tracking services.
- **No network access**: TidyBox operates completely offline. It does not connect to the internet.
- **No telemetry**: No usage data, crash reports, or diagnostics are sent to us or any third party.

### Data Stored on Your Device

TidyBox stores the following data **locally on your Mac only**:

- **Application settings**: Your preferences (watched directories, rules, notification settings, language preference).
- **Action history**: A log of file organization actions performed, stored in a local SwiftData database.
- **Security-Scoped Bookmarks**: Persistent access tokens for directories you have explicitly selected.

This data never leaves your device and is stored within the app's container.

### File Access

TidyBox accesses files and directories that you explicitly select through the system file picker. It uses macOS Security-Scoped Bookmarks to maintain access to these directories across app restarts. TidyBox does not access any files or directories that you have not explicitly granted permission to.

### Third-Party Services

TidyBox does not integrate with any third-party services, APIs, or SDKs.

### Children's Privacy

TidyBox does not collect any data from anyone, including children under 13.

### Changes to This Policy

If we update this privacy policy, we will post the changes on this page with an updated date.

### Contact

If you have any questions about this privacy policy, please create an issue at:
[https://github.com/kani-cream/tidybox-support/issues](https://github.com/kani-cream/tidybox-support/issues)

## 日本語

### 概要

TidyBox は macOS 向けのファイル自動整理アプリケーションで、完全にお使いのデバイス上で動作します。お客様のプライバシーの保護に努めています。

### データの収集

**TidyBox は個人データの収集、送信、保存を一切行いません。** 具体的には：

- **分析なし**: アナリティクスやトラッキングサービスは使用していません。
- **ネットワークアクセスなし**: TidyBox は完全にオフラインで動作します。インターネットに接続しません。
- **テレメトリなし**: 使用データ、クラッシュレポート、診断情報を当社や第三者に送信しません。

### デバイス上に保存されるデータ

TidyBox は以下のデータを **お使いの Mac のローカルにのみ** 保存します：

- **アプリケーション設定**: 設定内容（監視ディレクトリ、ルール、通知設定、言語設定）。
- **操作履歴**: 実行されたファイル整理操作のログ。ローカルの SwiftData データベースに保存。
- **Security-Scoped Bookmarks**: 明示的に選択されたディレクトリへの永続アクセストークン。

これらのデータはデバイスから外部に送信されることはなく、アプリのコンテナ内に保存されます。

### ファイルアクセス

TidyBox はシステムのファイルピッカーを通じてお客様が明示的に選択したファイルとディレクトリにのみアクセスします。macOS の Security-Scoped Bookmarks を使用して、アプリ再起動後もこれらのディレクトリへのアクセスを維持します。明示的にアクセスを許可していないファイルやディレクトリにはアクセスしません。

### サードパーティサービス

TidyBox はサードパーティのサービス、API、SDK と連携しません。

### お子様のプライバシー

TidyBox は13歳未満のお子様を含め、誰からもデータを収集しません。

### ポリシーの変更

このプライバシーポリシーを更新する場合は、更新日を添えてこのページに変更内容を掲載します。

### お問い合わせ

このプライバシーポリシーに関するご質問は、以下から Issue を作成してください：
[https://github.com/kani-cream/tidybox-support/issues](https://github.com/kani-cream/tidybox-support/issues)
