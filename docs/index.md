# TidyBox ユーザーガイド / TidyBox User Guide

## 日本語

TidyBox は、Mac のメニューバーに常駐するファイル自動整理アプリケーションです。指定したフォルダをリアルタイムに監視し、ルールに基づいてファイルの移動、リネーム、タグ付けを行います。完全オフラインで動作し、Apple Silicon Mac に最適化されています。

- [はじめに](user-guide/getting-started) -- インストールと初回セットアップ
- [監視ディレクトリ](user-guide/watched-directories) -- フォルダの追加と管理
- [ノードエディタ](user-guide/node-editor) -- ビジュアルルールの作成
- [プリセット](user-guide/presets) -- テンプレートルールの活用
- [ダッシュボード](user-guide/dashboard) -- 整理状況の確認
- [履歴](user-guide/history) -- 操作の確認と取り消し
- [アーカイブ](user-guide/archive) -- 古いファイルの段階的な管理
- [ゴミ箱管理](user-guide/trash) -- ゴミ箱のクリーンアップ
- [設定](user-guide/settings) -- 各種設定の調整
- [よくある質問](user-guide/faq) -- トラブルシューティング

## English

TidyBox is a macOS menu bar app that automatically organizes your files. It monitors folders in real time and moves, renames, and tags files based on rules you define -- all completely offline, optimized for Apple Silicon.
