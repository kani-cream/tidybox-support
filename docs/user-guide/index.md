# TidyBox ユーザーガイド / TidyBox User Guide

## 日本語

TidyBox は、Mac のメニューバーに常駐するファイル自動整理アプリケーションです。指定したフォルダをリアルタイムに監視し、ルールに基づいてファイルの移動、リネーム、タグ付けを行います。完全オフラインで動作し、Apple Silicon Mac に最適化されています。

- [はじめに](getting-started.md) -- インストールと初回セットアップ
- [監視ディレクトリ](watched-directories.md) -- フォルダの追加と管理
- [ノードエディタ](node-editor.md) -- ビジュアルルールの作成
- [プリセット](presets.md) -- テンプレートルールの活用
- [ダッシュボード](dashboard.md) -- 整理状況の確認
- [履歴](history.md) -- 操作の確認と取り消し
- [アーカイブ](archive.md) -- 古いファイルの段階的な管理
- [ゴミ箱管理](trash.md) -- ゴミ箱のクリーンアップ
- [設定](settings.md) -- 各種設定の調整
- [よくある質問](faq.md) -- トラブルシューティング

## English

TidyBox is a macOS menu bar app that automatically organizes your files. It monitors folders in real time and moves, renames, and tags files based on rules you define -- all completely offline, optimized for Apple Silicon.
